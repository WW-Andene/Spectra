## Audit summary

Spectra ships these surfaces today:
- IR camera-capture learning (rolling-shutter)
- Brute force IR protocol discovery
- Bundled IR remote DB (brand picker)
- Acoustic + RF + EM passive fingerprint scan
- Saved device library (per-file JSON in app private storage)
- Macros (multi-device command sequences)
- Universal-remote button grid screen
- Single-device clipboard share

What's missing relative to (a) baseline data-safety expectations,
(b) competing universal-remote apps (AnyMote, ZaZa, IR Universal), and
(c) Spectra's own innovation surface (passive multi-spectrum ID,
camera capture, RF discovery currently dead-ending in the matcher).

## Prioritization rationale

### Tier 1 — user value (B-001 → B-003)

- **B-001 Library backup**: A user who learns 20 remotes over months
  loses everything on phone replacement. There is no other escape
  hatch except clipboard-share-each-device-individually. This is
  data-continuity table stakes.
- **B-002 Widget**: Most-used remote command from the launcher without
  app-open friction. AnyMote ships this; Spectra doesn't.
- **B-003 Quick Settings tile**: Same idea, different surface.
  One-handed, even from lock screen. Cheap to add.

### Tier 2 — competitive parity (B-004 → B-008)

- **B-004 RF control**: Spectra already discovers WiFi/BLE/mDNS
  devices but only ever sends IR. A discovered Roku / WiFi smart
  bulb is a pure setup-input today; turning that into actual control
  is the natural next step and matches what every "smart home"
  remote app does.
- **B-005 Scheduling**: "Turn off TV at 11pm". Bedside fans.
- **B-006 Custom layouts**: Other apps let users rearrange buttons.
  Spectra ships a fixed grid.
- **B-007 NFC triggers**: Manifest already declares the permission +
  feature but no code uses it. Gap closure.
- **B-008 All-off**: Universal "everything off" macro generated
  automatically from devices with a `power` command.

### Tier 3 — innovation (B-009 → B-011)

- **B-009 Community fingerprints**: Spectra's acoustic+EM+RF passive
  signature is unique among universal remotes. Sharing those (anon)
  improves matching for everyone — Spectra-unique flywheel.
- **B-010 Auto-discover-and-control**: Walk into a room, app shows
  a control surface for everything it sees. Combines all four
  modules — Spectra-unique.
- **B-011 Voice / Assistant**: Lower priority because the platform
  hooks (Slices, App Actions) are slow-moving and the Tasker plugin
  intent path covers the power-user niche cheaply.
